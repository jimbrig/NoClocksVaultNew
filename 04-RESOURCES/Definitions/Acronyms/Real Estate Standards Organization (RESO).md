# Real Estate Standards Organization (RESO)

```table-of-contents
```

## Overview

**Real Estate Standards Organization (RESO)** is a non-profit trade organization that develops and promotes open standards for real estate data exchange and interoperability. RESO standards enable consistent data sharing between MLSs, brokerages, technology vendors, and other real estate industry participants.

## Key Concepts

**RESO Data Dictionary** defines standardized field names and values for property data.
**RESO Web API** specifies RESTful API patterns for real estate data access.
**RESO Certification** validates compliance with RESO standards.
**Universal Property Identifier (UPI)** is a standard for uniquely identifying properties.
**RETS** (Real Estate Transaction Standard) was the predecessor API standard.
