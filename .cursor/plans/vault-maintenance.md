# Vault Maintenance Plan

Regular maintenance tasks for the No Clocks Knowledge Vault.

## Weekly Tasks

1. **Process Inbox**
   - Review all notes in 00-INBOX/
   - Categorize and move to appropriate folders
   - Add proper frontmatter and tags

2. **Review WIP Notes**
   - Check notes with Status/WIP tag
   - Update or complete as needed
   - Move to Status/Complete when done

3. **Update MOCs**
   - Verify Dataview queries are working
   - Add new notes to relevant MOCs
   - Check for orphaned notes

## Monthly Tasks

1. **Tag Audit**
   - Review tag usage consistency
   - Consolidate similar tags
   - Remove unused tags

2. **Link Health**
   - Check for broken wikilinks
   - Update renamed note references
   - Add missing backlinks sections

3. **Archive Review**
   - Move outdated content to 99-ARCHIVES/
   - Update publish status as needed

## Quarterly Tasks

1. **Template Review**
   - Update templates with improvements
   - Add new templates as needed
   - Document template changes

2. **Quartz Sync**
   - Verify publishing workflow
   - Check site for broken links
   - Update ignored patterns if needed

3. **Backup Verification**
   - Confirm Git history is intact
   - Verify branch synchronization
   - Test restore process
