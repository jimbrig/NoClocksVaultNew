# Windows Tools

## Package Managers

- [WinGet](https://github.com/microsoft/winget-cli) - A command-line package manager for Windows.
  - [WinGet GUI](https://github.com/microsoft/winget-gui) - A graphical user interface for WinGet.
  - [Winstall](https://winstall.app/) - A web-based tool to generate WinGet installation scripts for multiple applications.

- [Chocolatey](https://chocolatey.org/) - A package manager for Windows that simplifies the installation and management of software.
  - [Boxstarter](https://boxstarter.org/) - A tool to automate the installation of software using Chocolatey.
  - [choco-cleanup](https://chocolatey.org/packages/choco-cleanup) - A tool to clean up old versions of Chocolatey packages.
  - [choco-package-list-backup](https://chocolatey.org/packages/choco-package-list-backup) - A tool to back up and restore a list of installed Chocolatey packages.
  - [Chocolatey GUI](https://chocolatey.org/packages/ChocolateyGUI) - A graphical user interface for Chocolatey.
  - [Chocolatey Package Explorer](https://chocolatey.org/packages/ChocolateyPackageExplorer) - A tool to create and edit Chocolatey packages.
  - [Chocolatey Updater](https://chocolatey.org/packages/ChocolateyUpdater) - A tool to automatically update Chocolatey packages.

- [Scoop](https://scoop.sh/) - A command-line installer for Windows that focuses on simplicity and ease of use.
  - [scoop-search](https://github.com/ScoopInstaller/scoop-search) - A tool to search for applications in Scoop buckets.
  - [Scoop Apps](https://scoop-apps.github.io/) - A collection of additional Scoop buckets for more applications.
  - [Scoop GUI](https://github.com/ScoopInstaller/ScoopGUI) - A graphical user interface for Scoop.
  - [Scoop Directory](https://rasa.github.io/scoop-directory/) - A directory of Scoop buckets and applications.
  - [shovel.sh](https://shovel.sh/) - A web-based tool to search and install Scoop applications.

More Legacy or Outdated Package/Software Management Tools:

- [Ninite](https://ninite.com/) - A tool to automate the installation of multiple applications on Windows.
- [OneGet](https://docs.microsoft.com/en-us/powershell/module/oneget/?view=powershell-7.1) - A package management framework for Windows PowerShell.
- [AppGet](https://appget.net/) - A package manager for Windows that allows users to install and manage software from the command line.
- [PatchMyPC](https://patchmypc.com/home-updater) - A tool to keep installed applications up to date on Windows.
