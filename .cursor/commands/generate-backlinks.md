# Generate Backlinks Section

Add or update the backlinks section in a note using Dataview.

## Instructions

1. Check if the note has a backlinks section
2. If missing, add the standard backlinks query
3. If present, verify the query is correct

## Standard Backlinks Section

Add this at the end of notes:

```markdown
### Backlinks
<!-- dynamic content -->
```

## Placement

The backlinks section should be:
- Under an Appendix heading if one exists
- At the very end of the note
- After See Also section if present
