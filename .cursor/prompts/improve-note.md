# Improve Note

Analyze and improve an existing Obsidian note.

## Instructions

1. Review the note's structure and content
2. Check frontmatter completeness
3. Identify missing elements
4. Suggest improvements

## Analysis Checklist

Frontmatter:
- [ ] Has creation_date and modification_date
- [ ] Has appropriate Status tag
- [ ] Has appropriate Type tag
- [ ] Has Topic tags
- [ ] Has description field
- [ ] Has aliases if applicable
- [ ] Has publish field

Content:
- [ ] Has clear heading structure
- [ ] Has overview/introduction
- [ ] Uses wikilinks for internal references
- [ ] Has See Also section
- [ ] Has Backlinks dataview query
- [ ] Code blocks have syntax highlighting

## Output Format

Provide:
1. List of identified issues
2. Suggested improvements
3. Updated content if requested
