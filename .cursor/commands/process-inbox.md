# Process Inbox

Process notes from the inbox folder and organize them into appropriate locations.

## Instructions

1. List all notes in `00-INBOX/`
2. For each note:
   - Analyze content to determine appropriate category
   - Suggest target folder based on content type
   - Ensure frontmatter is complete
   - Add appropriate tags
3. Provide a summary of recommended moves

## Folder Mapping

| Content Type | Target Folder |
|--------------|---------------|
| Atomic concepts | 01-SLIPBOX/ |
| Project notes | 02-PROJECTS/{ProjectName}/ |
| Journal entries | 03-AREAS/Journal/{Year}/ |
| Definitions | 04-RESOURCES/Definitions/ |
| Code snippets | 04-RESOURCES/Code/{Language}/ |
| Articles | 04-RESOURCES/Articles/ |
| Reference lists | 04-RESOURCES/Lists/ |

## Output

For each inbox note, provide:
1. Current filename
2. Recommended destination
3. Suggested tag additions
4. Any frontmatter updates needed
