# List - Property Data Platforms
## Overview

Curated collection of **property data API platforms, parcel data providers, and real estate data services** for integrating property intelligence into applications.

## National Parcel Data Providers

| Platform | Description | Links |
|----------|-------------|-------|
| Regrid | National parcel data with 155M+ parcels, zoning, ownership, boundaries | [Website](https://regrid.com) - [API Docs](https://support.regrid.com/api) |
| LightBox | Property data platform with parcel, assessment, flood data APIs | [Website](https://lightboxre.com) - [Docs](https://lightbox.document360.io) |
| ATTOM Data | AI-ready property data from public records and geospatial sources | [Website](https://attomdata.com) - [API](https://api.gateway.attomdata.com) |
| CoreLogic | Property data, analytics, valuations, risk assessment | [Website](https://corelogic.com) |
| First American | Title, property, and risk data solutions | [Website](https://firstam.com) |
| Black Knight (ICE) | Property data and mortgage analytics | [Website](https://icemortgagetechnology.com) |
| RealEstateAPI | Property data API for search, comps, and valuation | [Website](https://realestateapi.com) - [Docs](https://developer.realestateapi.com) |
| Acres | Geospatial land research with 150M+ parcels, ag productivity data | [Website](https://acres.com) |
| Realie.ai | AI-normalized property data with real-time updates and S3 bulk exports | [Website](https://realie.ai) - [Docs](https://docs.realie.ai) |
| Land ID | 40+ data overlays with comprehensive boundary and ownership data | [Website](https://id.land) |
