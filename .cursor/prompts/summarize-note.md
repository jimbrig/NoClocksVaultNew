# Summarize Note

Generate a concise summary of an Obsidian note's content.

## Instructions

1. Read the provided note content
2. Extract key points and main ideas
3. Generate a bullet-point summary
4. Optionally update the note's description frontmatter field

## Output Format

Provide:
1. A suggested `description` field value (one sentence)
2. A bullet-point summary of key takeaways
3. Suggested related topics or links if applicable

## Guidelines

- Keep summaries focused and actionable
- Preserve technical accuracy
- Identify potential wikilinks to other vault notes
- Suggest relevant tags if missing
